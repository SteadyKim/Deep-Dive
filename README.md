## What This Repository Covers

- 프로젝트 진행 중 접한 핵심 CS 개념
- 개념의 정의와 동작 원리

## Why This Repository Exists

즉, 이 저장소의 목적은 다음과 같습니다.

- 프로젝트 경험을 CS 학습으로 연결하기
- 학습 내용을 누적 가능한 형태로 정리하기
